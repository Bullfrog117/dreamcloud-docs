================================================ How to create static web sites with DreamObjects ================================================ Create the bucket ----------------- The name of the bucket should be the same name of the CNAME. For example, if your website will be example.com, name the bucket example.com. [user@localhost]$ s3cmd mb s3://example.com Bucket 's3://example.com/' created Set the ACL for the bucket -------------------------- [user@localhost]$ s3cmd setacl --acl-public s3://example.com s3://example.com/: ACL set to Public Create the website for the bucket --------------------------------- [user@localhost]$ s3cmd ws-create --ws-index index.html s3://example.com Bucket 's3://example.com/': website configuration created. Set the policy for the bucket and add static files -------------------------------------------------- [user@localhost]$ s3cmd -Pr sync . s3://example.com -P set all content public -r recursive copy Done: http://example.com.objects-website-us-west-1.dream.io/ Create the CNAME ---------------- Go to the `Panel -> Domains `_ and add a custom DNS record to the domain. Pick `CNAME` and add example.com.objects-website-us-west-1.dream.io to empty the bucket aws s3 rm --recursive s3://example.com --endpoint-url https://objects-us-west-1.dream.io
