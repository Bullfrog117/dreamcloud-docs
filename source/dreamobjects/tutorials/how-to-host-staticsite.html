================================================ How to create static web sites with DreamObjects ================================================ Create the bucket ----------------- The name of the bucket should be the same name of the CNAME. For example, if your website will be example.com, name the bucket example.com. [user@localhost]$ s3cmd mb s3://www.example.com Bucket 's3://www.example.com/' created Create the website for the bucket --------------------------------- [user@localhost]$ s3cmd ws-create --ws-index index.html s3://www.example.com Bucket 's3://www.example.com/': website configuration created. Set the policy for the bucket and add static files -------------------------------------------------- [user@localhost]$ s3cmd -Pr sync . s3://www.example.com -P set all content public -r recursive copy Done: http://www.example.com.objects-website-us-west-1.dream.io/ Create the CNAME ---------------- Go to the `Panel -> Domains `_ and add a custom DNS record to the domain. Pick `CNAME` and insert `www.example.com.objects-website-us-west-1.dream.io`.
